A layout exercise collecting images of contemporary graphic design and pairing it with body copy taken from articles about design. The goal was to understand how to create a cohesive system and consistent but engaging rythm to the publication.

I focused on breaking the rules of the typical publication layout. Instead of vertically set body copy, I set it horizontally, instead of small page numbers on the side, I made them bold, front and center.

The book also read two ways. The arrows on the sides of the page indicate the direction in which to read the text which ended up becoming the inspiration for the book cover. The colored arrows correspond with the text set in the same color.
